JMeter测试持续集成

所需软件

JDK、JMeter、Ant、Tomcat、Jenkins 。

编写脚本

写JMeter脚本，把jmx文件放在一个脚本文件夹里。

用Ant执行脚本

1、首先将 JMeter的extras目录中ant-jmeter-1.1.1.jar包拷贝至Ant安装目录下的lib目录中，其次修改JMeter的bin目录下jmeter.properties文件的配置：jmeter.save.saveservice.output_format=xml 。

2、编写build.xml，写JMeter目录、jmx脚本目录、报告名称、报告路径、报告压缩打包 。

3、执行build.xml，cmd进入build.xml所在目录，输入：ant ，即可 。

集成到Jenkins

1、把Jenkins.war包放在Tomcat目录的webapps下，启动Tomcat 。

2、在浏览器中输入http://localhost:8080/jenkins/ ，访问Jenkins ，端口8080可以设置的，在conf文件夹下的server.xml中设。

3、在Jenkins，添加Performance plugin 、 HTML Publisher plugin 、ant插件，新建一个项目，配置项目信息：

  • 项目名称（全英文，不带空格），构建一个自由风格的软件项目、Invoke Ant,写build.xml路径，Targets：all ，Build File:build.xml路径。
  • 在Publish HTML，HTML directory to archive：D:\Program Files\jmeter\workspace\report\html；Index page[s]：TestReport.html,report-details.html
  • 在JMeter Report files，D:\Program Files\jmeter\workspace\report\jtl\TestReport.jtl，转化JMeter报告jtl文件，在Jenkins显示 。
  • 在Editable Email Notification中，设置发送邮件的信息，包括构建信息，发送测试结果的报告， Attachments：*/.zip。
