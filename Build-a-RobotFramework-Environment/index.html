搭建RobotFramework的环境

1.Windows下安装Python

首先访问官网下载需要的python版本。

32位系统下载Windows x86 MSI installer，64位系统下载Windows x86-64 MSI installer 。

安装下载包，双击msi文件，按提示安装即可。

把python的安装目录添加到环境变量->系统变量Path中。

验证是否安装成功,在cmd中输入命令行：

2.Linux下安装Python

首先访问官网下载需要的python版本。

或在Linux服务器下下载安装包，wget “https://www.python.org/ftp/python/2.7.14/Python-2.7.14.tgz“

解压安装包，需要root权限，tar -zxvf Python-2.7.14.tgz

解压后进入Python-2.7.14目录,cd Python-2.7.14

安装,输入./configure

补:configure命令执行完之后，会生成一个Makefile文件，这个Makefile主要是被下一步的make命令所使用。打开Makefile你就会发现，里边制定了构建的顺序，Linux需要按照Makefile所指定的顺序来构建(build)程序组件。

执行命令make，make实际上编译你的源代码，并生成执行文件。

再执行make install命令

make install实际上是把生成的执行文件拷贝到linux系统中必要的目录下，比如拷贝到/usr/local/bin目录下，这样所有user就都能运行这个程序了。

到这里Python就算安装完成了。需要说的是，其实其他的linux软件安装也是大同小异的，基本都需要走configure>make>make install这么一个过程。

把安装的python变成默认版本，执行python命令，仍旧是指向旧的版本

在命令行输入python2.7,这样就会执行到我们新的python版本了。

3.安装RobotFramework

官方地址：http://robotframework.org/

pip安装:
打开python的安装目录，会发现该目录下存在一个叫Scripts的文件夹，点开Scripts，会发现里面有一系列和easy_install有关的文件。

cd到easy_install.exe的根目录下，执行命令：easy_install.exe pip

则Scripts文件夹下会出现一系列和pip有关的文件，其中有pip.exe

将pip.exe的路径也添加到环境变量PATH中，在cmd下输入“pip”，如果能识别”pip”指令，则说明pip安装成功了。

pip安装RobotFramework:
执行命令：pip install robotframework

在cmd中输入命令行验证是否安装成功：pybot –version，如成功显示版本号，则表示robot安装成功。

4.安装wxPython

注意：安装wxPython.exe注意ride不支持2.8.12.1以上的。

http://sourceforge.net/projects/wxpython/files/wxPython/2.8.12.1/

32位系统装win32,64位系统装win64。

5.安装Ride

手动： https://pypi.python.org/pypi/robotframework-ride

命令： pip install robotframework-ride

然后到 python安装目录/Scripts 目录下，点击ride.py即可。

ride.py 运行是会闪动一会命令行窗口，如果不想出现这个窗口。可以把 ride.py 改为 ride.pyw，就解决了。

6.添加库

selenium库： pip install –upgrade robotframework-seleniumlibrary

建议火狐浏览器版本不要超过46
